# 功能演示

## 1. Markdown 渲染效果

AI 的回复会自动渲染为 Markdown 格式，支持：

### 标题
```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 代码块
````markdown
```javascript
function hello() {
  console.log('Hello World!');
}
```
````

会渲染为带语法高亮的代码块：
```javascript
function hello() {
  console.log('Hello World!');
}
```

### 列表
```markdown
- 无序列表项 1
- 无序列表项 2
  - 嵌套项

1. 有序列表项 1
2. 有序列表项 2
```

### 表格
```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 数据1 | 数据2 | 数据3 |
```

### 引用
```markdown
> 这是一段引用文本
> 可以多行
```

### 链接和图片
```markdown
[链接文本](https://example.com)
![图片描述](https://example.com/image.png)
```

## 2. 右键菜单功能

### 复制消息
1. 在任意消息上右键点击
2. 选择"复制"
3. 消息内容会复制到剪贴板
4. 可以粘贴到其他应用中

**使用场景：**
- 保存 AI 的回答
- 分享对话内容
- 导出重要信息

### 删除消息
1. 在任意消息上右键点击
2. 选择"删除"
3. 该条消息会从对话历史中移除

**使用场景：**
- 删除错误的提问
- 清理无关的对话
- 重新组织对话流程

### 示例 2：代码示例请求

**提问：**"给我一个使用 asyncio 的简单示例"

**AI 可能的回答：**
````markdown
这是一个简单的 asyncio 示例：

```python
import asyncio

async def main():
    print('Hello')
    await asyncio.sleep(1)
    print('World')

asyncio.run(main())
```

这个示例展示了：
1. 使用 `async def` 定义异步函数
2. 使用 `await` 等待异步操作
3. 使用 `asyncio.run()` 运行主函数
````

代码会自动高亮显示，非常清晰！

## 4. 消息管理技巧

### 技巧 1：清理无关对话
当对话偏离主题时：
1. 右键点击无关消息
2. 选择"删除"
3. 继续主题对话

### 技巧 2：保存重要回答
当 AI 给出有价值的回答时：
1. 右键点击该消息
2. 选择"复制"
3. 粘贴到笔记应用中保存

### 技巧 3：重新提问
如果 AI 的回答不满意：
1. 删除 AI 的回答
2. 删除你的提问
3. 重新组织问题再次提问

## 6. 最佳实践

### 提问技巧
1. **具体明确**：说明你想要什么格式的回答
   - ❌ "介绍一下这个库"
   - ✅ "用表格总结这个库的主要 API"

2. **利用 Markdown**：要求 AI 使用特定格式
   - "用代码块展示示例"
   - "用列表列出步骤"
   - "用表格对比差异"

3. **分步骤**：复杂问题分解为多个小问题
   - 先问概述
   - 再问细节
   - 最后要示例

### 对话管理
1. **定期清理**：删除无关消息保持对话清晰
2. **保存重要内容**：及时复制有价值的回答
3. **重新开始**：话题切换时清空对话历史

### 网页选择
1. **官方文档优先**：信息最准确
2. **内容完整**：避免过于简短的页面
3. **格式规范**：结构清晰的网页解析效果更好

## 7. 快捷键

- `Ctrl + Enter`：发送消息
- 右键点击消息：打开上下文菜单
- 点击空白处：关闭上下文菜单

## 8. 注意事项

1. **Markdown 渲染仅限 AI 回复**：用户消息保持纯文本显示
2. **代码高亮需要指定语言**：在代码块中标注语言类型
3. **删除消息不可恢复**：谨慎使用删除功能
4. **复制包含格式**：复制的是 Markdown 源码，不是渲染后的 HTML
